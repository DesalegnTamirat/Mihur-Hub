# Mihur-Hub

Mihur-Hub is a collaborative workspace for software engineering teams where an AI assistant lives inside your project and actually understands your codebase. 

Instead of switching between your code editor, a chat tool, a documentation wiki, and a separate AI tool, your team has one place where all of that converges — and the AI in that place has been trained on your actual project files, not just generic programming knowledge.
